# website-of-eInkBoard
eInkBoard 是使用中天微 CH2201 “赤焰剑” 开发板作为主控的 [NESDC 2018](http://www.socchina.net/) 参赛作品，本 repo 是当时这个作品的网页的归档。点击[此处](https://csy-tvgo.github.io/website-of-eInkBoard/)浏览。  

> 2021 Jun 08 更新: 我们复刻了 eInkBoard 这一经典之作，用于显示 [VeriMake 哔哩哔哩账号](https://space.bilibili.com/356383684) 的实时粉丝数量，取名为 eInkBoard v2，完整代码已开源于 [这个 repo](https://github.com/CSY-tvgo/eInkBoard-v2)。
